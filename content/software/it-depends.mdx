It Depends is a small application build to give you insights into your Software Bill of Materials or SBOM for short. With this application you can visualize all the dependencies based on the generated SBOM, and give you insights in the versions that are used.

It Depends main focus is allowing a developer to get a grip on the dependencies and transative dependencies of their system. It Depends supports different kinds of SBOM standards. Based on these standards you can load the generated SBOM file into It Depends and it will visualize all the dependencies for you.
