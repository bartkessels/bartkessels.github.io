GetIt is a native HTTP client designed specifically for the Linux desktop. Built with GTK4 and libadwaita, it provides a beautiful, modern interface that integrates seamlessly with GNOME.

## Features

- **Native Experience**: Built with GTK4 for a true Linux-native feel
- **Request Collections**: Organize your API requests into collections
- **Environment Variables**: Manage different environments easily
- **Code Generation**: Export requests to various programming languages
- **Dark Mode**: Full support for light and dark themes

## Tech Stack

- GTK4 / libadwaita
- C / GLib
- Meson build system
