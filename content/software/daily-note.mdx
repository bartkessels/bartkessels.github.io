Daily Note is a minimalist journaling app that encourages you to write one note each day. No folders, no tags, no complexity - just your thoughts, organized by date.

## Philosophy

Most note apps overwhelm you with features. Daily Note takes the opposite approach: simplicity. One note per day forces you to be intentional about what you write.

## Features

- One note per day
- Calendar view to browse past notes
- Full-text search
- Material You theming
- Local-only storage (your data stays on your device)
- No account required

## Tech Stack

Built with modern Android technologies:
- 100% Kotlin
- Jetpack Compose for UI
- Room for local database
- Material Design 3
