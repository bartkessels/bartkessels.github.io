Something I think is required to be known, all posts on this website are completely written by me and checked by a LLM to make sure my sentences are correct and legible. Every task that I can't do and doesn't require any creativeness is done using Github Copilot.

To be completely transparent, these are the tasks I delegate to various AI-tools:

- Generating images for each post
    - __AI-tool__: Apple Image Playground
- Checking if my posts are well structured and not missing anything
    - __AI-tool__: Github Copilot using `GPT-5.4 (High)`
- Maintaining the code of this website
    - __AI-tool__: Github Copilot using `Claude Sonnet 4.6`

The initial setup of this website was done using Vercel's V0 model and further implementation was done using Github Copilot with `Claude Sonnet 4.6` and a lot of manual changes. The manual changes were mostly done because of a specific structure I wanted into my codebase, mainly the MVVM pattern and some other tweaks in the code. For more repetitive tasks, such as updating all the imports to use single quotes instead of double quotes, I put the LLM to work.
